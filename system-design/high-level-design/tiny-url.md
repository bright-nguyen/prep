# Tiny URL (URL Shortener)
## Requirements
- Functional Requirements
  - Given a long URL create a short URL
  - Given a short URL redirect to a long URL
- Non Functional Requirements
  - Very low latency
  - Very high availability
- Not covered
  - Updating of URLs
  - Deleting of URLs
